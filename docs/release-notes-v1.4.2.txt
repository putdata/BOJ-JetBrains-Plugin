## v1.4.2 - SDK 선택 & 버그 수정

### SDK 선택 기능
Java/Python SDK를 IDE 내에서 직접 선택하여 컴파일·실행할 수 있습니다.
- IntelliJ에 등록된 Java SDK, Python 인터프리터 목록에서 선택
- 언어별 마지막 선택 SDK 자동 기억
- BOJ 테스트 패널 및 일반 테스트 패널에서 모두 사용 가능

### 버그 수정
- BOJ Helper 도구 창 기본 위치를 오른쪽에서 왼쪽으로 변경
- 보일러플레이트 경로 템플릿 {title} 변수 미치환 수정 및 문제 번호 고정
